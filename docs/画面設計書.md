### S3. パーティ編集（カバレッジパネル追加）
```
+----------------------------------------------------+
| パーティ名: [ メイン構築___________ ]              |
| 1〜6: [型選択ドロップダウン × 6]                   |
|                                                    |
| メガシンカ: 1 / 優先: 0                            |
+----------------------------------------------------+
| カバレッジ確認     [上位20] [上位30] を2匹カバー判定|
| ✅12  🔵3  ⚠️5  ❌2                                |
|                                                    |
| ❌ 未カバー (2匹)                                  |
|   [ガブリアス] [カイリュー]                        |
|                                                    |
| ⚠️ △のみ (5匹)                                     |
|   [ドラパルト] ...                                 |
|                                                    |
| 🔵 1匹カバー (3匹)                                 |
|   [ゲンガー] ...                                   |
|                                                    |
| ▶ ✅ カバー済み (12匹) ← details折りたたみ         |
+----------------------------------------------------+
```

カバレッジ判定ロジック:
- TOP50中の上位N匹（Nは閾値ボタンで20/30を切替）: 2匹以上○で `full`、1匹○で `single`
- 残りの匹: 1匹以上○で `full`
- ○なし△あり: `partial`
- 全×: `none`

### S5. 型編集（ソートボタン追加）
有利不利テーブルのツールバー:
```
フィルタ: [___________]  並び: [図鑑番号順] [使用率順]
```
- 一括ボタン（○△×クリア）は削除済み
- 使用率順: ALTEMA TOP50順（ランク外は末尾に図鑑番号順で表示）
