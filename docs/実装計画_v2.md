# 実装計画 v2（現状版）

## 実際の技術構成（確定）
| 項目 | 実際の選定 | 備考 |
|---|---|---|
| 言語 | HTML / CSS / JavaScript (バニラ) | Vue 3は当初案だったが不採用 |
| スタイル | 素のCSS（CSSカスタムプロパティ） | Tailwindも不採用 |
| データ永続化 | localStorage + Firebase Firestore | Phase 3で実装済み |
| ホスティング | GitHub Pages | https://otowa0go.github.io/poke/ |
| ポケモンデータ | data.js（1052エントリインライン） | PokeAPI由来、リージョンフォーム含む |

## 実際のディレクトリ構成
```
poke/
├── index.html
├── manifest.json          # PWA用
├── service-worker.js      # PWA用（オフラインキャッシュ）
├── css/
│   └── style.css
├── js/
│   ├── app.js             # ルーター・初期化
│   ├── data.js            # ポケモンマスタ 1052匹 + USAGE_RANK
│   ├── store.js           # localStorage永続化 + Firestore同期
│   ├── search.js          # 予測入力
│   ├── score.js           # スコアリング
│   ├── kana.js            # ひらがな正規化
│   ├── pokemon-input.js   # 予測入力UIコンポーネント
│   └── views/
│       ├── home.js
│       ├── type-list.js
│       ├── type-edit.js   # 有利不利登録 + ソート機能
│       ├── party-list.js
│       ├── party-edit.js  # カバレッジパネル付き
│       ├── simulator.js
│       └── settings.js    # インポート/エクスポート
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
├── data/
│   ├── pokemon_all.json   # 全ポケモンデータ（参照用）
│   └── usage_ranking.json # ALTEMA使用率TOP50
├── scripts/
│   ├── fetch-pokemon.js   # PokeAPI取得スクリプト（一度きり実行用）
│   └── fetch-forms.js     # リージョンフォーム取得スクリプト
├── enable_poke/           # 使用可能ポケモン更新用
│   ├── 001_enable_poke_M_A.json   # Champions console抽出データ
│   └── ポケモン取得用コード.txt    # 抽出用コード
├── docs/
└── backup/
```
