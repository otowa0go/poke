# 基本仕様書 - ポケモン対戦補助ツール (ポケモンチャンピオンズ用)

## 1. 概要
ポケモンチャンピオンズの対戦における選出補助ツール。
自分のパーティ6匹から、相手のパーティ6匹に応じて最適な選出3匹を自動で算出する。

## 2. 利用シーン
- 対戦前の選出時間（1分30秒）以内に、相手6匹に対する最適選出3匹を決定する
- パーティ作成時に、ポケモン型ごとの有利不利を登録・編集する

## 3. 機能一覧
| ID | 機能 | 説明 |
|---|---|---|
| F1 | ポケモン型管理 | 同じポケモンの異なる育成型を複数登録 |
| F2 | パーティ管理 | 6体の型でパーティを構成、複数保存可 |
| F3 | 有利不利登録 | 型ごとに相手ポケモン全種への ○/△/× を登録 |
| F4 | 選出シミュレート | 相手6匹を入力して最適3匹を計算・表示 |
| F5 | 設定管理 | スコア重み、優先ポケモン、略称辞書 |
| F6 | 使用可能ポケモンリスト管理 | 表示対象を絞り込む（インポート対応） |

## 4. 機能要件詳細

### F1. ポケモン型管理
- 同じ種類のポケモンを複数登録できる（内部IDで区別）
- 表示名: 「ポケモン名 (型名)」 例: 「ガブリアス (スカーフ)」
- 編集項目:
  - **ポケモン種**（必須、使用可能ポケモンから選択）
  - **型名/メモ**（任意、見分け用）
  - **持ち物**（フリー入力 1つ）
  - **技**（フリー入力 4つ）
  - **努力値** HP/攻撃/防御/特攻/特防/素早 各 0〜32 / 合計 66 まで
  - **性格**（25種から1つ選択）
  - **メガシンカフラグ**
  - **優先ポケモンフラグ**
  - **有利不利テーブル**（F3）

### F2. パーティ管理
- 1パーティ = 型ID × 6
- 複数パーティを作成・編集・削除・複製可
- パーティ一覧から1つを選択してシミュレートに使う

### F3. 有利不利登録
- ポケモン型ごとに、使用可能ポケモン全種に対して ○/△/× を登録
- 未登録は △ 扱い
- 一覧形式で素早く編集できるUI（フィルタ・一括設定）

### F4. 選出シミュレート（最重要機能）
**入力**
- 自分のパーティ（保存済みパーティから選択）
- 相手の6匹（種のみ。予測入力対応）

**出力**
- 上位候補 2〜5案、ランク順
- 各候補について以下を表示:
  - 順位アイコン（🥇🥈🥉）
  - 選出3匹の名前
  - 合計スコア（カバレッジ点 / 詳細点 / ボーナス点の内訳）
  - **「重い相手」**: ○がいない相手と、その×の数
  - メガシンカ数 / 優先ポケモン数

**スコアリング**
```
合計スコア = カバレッジ点 + 詳細点 + ボーナス点

[カバレッジ点] (相手1匹あたり、6匹分を合計)
  ○が1匹以上いる    → +10
  △が2匹以上いる    → +6
  △が1匹のみ        → +3
  全部×             → 0
  最大: 60点

[詳細点]
  選出3匹 × 相手6匹 = 18マスのマトリクスを集計
  ○: +2 / △: 0 / ×: -2

[ボーナス点]
  メガシンカポケモン × 1匹 → +α (デフォルト +5)
  優先ポケモン       × 1匹 → +β (デフォルト +5)
```
- すべての重み(10/6/3、+2/-2、α、β)は設定画面で変更可能

**候補の絞り込みロジック**
- 同点・近接スコア時は最大5候補を表示
- 上位スコアとの差が大きい候補は表示しない
- 1位を強調表示

### F5. 設定管理
- スコア重みカスタマイズ
- 略称辞書管理（追加/編集/削除）
- 優先ポケモン設定（型単位、F1で設定）
- 使用可能ポケモンON/OFF（チェックボックス一覧）

### F6. 使用可能ポケモンリスト管理
- リスト表示・トグル切替
- **インポート機能**: テキスト/CSV (1行1ポケモン名、英語/日本語対応)
- エクスポート機能
- 将来: 公式URLからの自動取得（公式が機械可読リストを提供したら対応）

## 5. 入力支援（予測入力）
- カタカナ・ひらがな・英語・略称・図鑑番号 すべて対応
- 例: 「がぶ」「ガブ」「Garchomp」「Garc」「445」 → ガブリアス
- 候補は1〜2文字目から表示（応答100ms以内）
- 略称辞書は初期辞書を同梱、ユーザー追加可

## 6. 非機能要件
| 項目 | 要件 |
|---|---|
| 対応端末 | PC（Windows/Mac）+ スマホ（iOS/Android） |
| ブラウザ | 主要モダンブラウザ |
| オフライン動作 | 可（PWA化を視野） |
| シミュレート性能 | 1秒以内（実際は数ms） |
| データ同期 | PC↔スマホでクラウド同期 |
| インストール | 不要（URLアクセスのみ） |

## 7. 制約・前提
- 努力値は **チャンピオンズ仕様（各0〜32 / 合計66）**
- 持ち物・技はフリー入力（マスタなし）
- 性格は固定マスタ（25種）
- 使用可能ポケモンの初期リストは要望.txtの186匹（※随時更新可能）
- 公式の機械可読な使用可能ポケモンリストは現状なし
